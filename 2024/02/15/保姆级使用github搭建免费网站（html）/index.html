保姆级使用github desktop搭建免费web网站的教程(简易版）

发表于2024-02-15|更新于2024-02-16|教程向
|阅读量:

前言：可能写的不是很清楚，欢迎指正

一：创建github账户

注册 https://github.com ，点击左上角的 sign up （登记/注册）
注意，用户名要记住，
PS：可能有时候无法访问或访问过慢，这是正常的，可能要开一个加速器。
注册需要有邮箱，可以直接使用QQ邮箱进行注册。
注册这里懒得截图演示了,可以参考B站教程
注册可以参考这个：https://www.bilibili.com/video/BV1Ty4y1K7Fh/?spm_id_from=333.337.search-card.all.click&vd_source=0b518babf53c69fcf8aee04061b42063

二：github desktop
      github desktop相对于git的好处就是有图形化的演示

创建好仓库后，下载 github desktop
可以在这里下载→ https://soft.3dmgame.com/down/279673.html

安装好后，大概是这样。点击sign into github.com
跳转github进行登录

进来以后应该是这样，点击左边的Create new repository，创建新的存储库

repository name用来命名
注意：这里命名格式一定要为

用户名.github.io

不然会出bug。

这里路径我是推荐直接在磁盘上创建，他后面会自己创建一个git文件夹
填好后，不用管其他的，继续就可以了
我这边以 text1 命名用来演示

可以看到在E盘多了一个text1的文件夹
在这个文件夹内，你可以编辑自己的网页，
这里有个网页的模板的链接可以参考：
https://html5up.net
下载以后，解压安装包，把安装包内的内容复制到 E:/text1即可
用游览器打开index.html文件，可以查看当前网站

上传到github

我这里懒，没怎么编辑

点击commit to main,意思是提交对于这个文件夹的更改
点击上方的publish repository,也就是发布库，
点击后，关闭keep this code private，意思是保持代码私密性
然后点Publish repository,发送

最后，游览器输入网址用户名.github.io就可以访问了
可以参考我一个朋友的网站：https://tikana0.github.io/
关于web网站该如何基于模板搭建，可以期待我下一篇博文。
